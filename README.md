## 🚀 Como publicar no GitHub Pages
1. Crie uma conta em https://github.com (grátis).
2. Crie um repositório novo (ex.: `kveiculos`).
3. Envie **todo o conteúdo desta pasta `site/`** para o repositório
   (o `index.html` precisa ficar na raiz do repositório).
4. No repositório, vá em **Settings → Pages**.
5. Em **Source**, escolha a branch `main` e a pasta `/ (root)`. Salve.
6. Aguarde ~1 minuto. Seu site estará no ar em:
   `https://SEU-USUARIO.github.io/kveiculos/`

> Dica: depois dá para ligar um domínio próprio (ex.: `kveiculos.com.br`)
> em **Settings → Pages → Custom domain**.

## 🔄 Como trocar/atualizar as fotos
- Substitua o arquivo dentro de `imagens/` mantendo **o mesmo nome**
  (ex.: troque `golf.webp` por outra foto chamada `golf.webp`).
- Não precisa mexer no código.
- Use fotos na horizontal, ~1200px de largura, em `.webp` ou `.jpg`
  (deixa o site leve e rápido no celular).
